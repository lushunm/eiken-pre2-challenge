# えいけん準2級チャレンジ！ 🦊

小学生向けの英検準2級学習ゲームです。ブラウザだけで遊べます。

**▶ あそぶ: https://lushunm.github.io/eiken-pre2-challenge/**

## モード

- 📖 **たんごクイズ** — 頻出単語64語（英語⇔意味、自動発音つき）
- ✏️ **ぶんぽうクイズ** — 穴うめ問題24問（やさしい解説つき）
- 💬 **かいわクイズ** — 会話の応答12問（音声読み上げつき）
- 🎧 **リスニング** — 音声合成による聞き取り問題16問
