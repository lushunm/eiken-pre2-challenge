# えいけん準2級チャレンジ！ 🦊

小学生向けの英検準2級学習ゲームです。ブラウザだけで遊べます。

**▶ あそぶ: https://lushunm.github.io/eiken-pre2-challenge/**

## モード

### トレーニング
- 📖 **たんご** — CEFR-J準拠の約2,300語（A2全域＋B1頻出）。英語⇔意味、自動発音つき
- 🧩 **じゅくご** — 頻出熟語・句動詞・構文 約300本（例文つき）
- ✏️ **ぶんぽう** — 準2級文法の穴うめ問題（やさしい解説つき）
- 💊 **ふくしゅう** — まちがえた問題を優先して再出題

### ほんばん形式（2024年度リニューアル後の公式出題形式を再現）
- 📝 **たんぶん** — 大問1形式：短文の語句空所補充
- 💬 **かいわぶん** — 大問2形式：会話文の文空所補充
- 📚 **ちょうぶん** — 大問4縮小版：パッセージ＋内容一致問題
- 🎧 **リスニング** — 公式と同じ3部構成（第1部 応答選択3択／第2部 会話一致4択／第3部 文一致4択）。音声合成で2話者を読み分け

### スペシャル
- 🏆 **もぎけん** — 本番ミニ模試30問（大問1→大問2→リスニング3部を公式の順で出題）
- 🎤 **めんせつ** — 二次試験の練習20セット（黙読→音読→質問→モデル回答）※イラスト問題は対象外
- 📒 **たんごちょう** — 検索・品詞タブ・にがてリスト・発音つき

### 収録ボリューム
単語2,346語（CEFR-J A2全域＋B1頻出）／熟語・構文295本／文法150問／
大問1形式150問／大問2形式60問／リスニング150問／長文20パッセージ51問／面接20セット
＝ **クイズ問題 約660問＋語彙2,600超**

## 内容について（大事なおことわり）

- 問題はすべて**オリジナル作問**です。英検の過去問の転載ではなく、日本英語検定協会とは関係ありません
- 出題形式は[公式サイト公開の試験形式](https://www.eiken.or.jp/eiken/exam/grade_p2/)（2024年度リニューアル後）を参考にしています
- 語彙の選定には『**CEFR-J Wordlist Version 1.5**』（東京外国語大学 投野由紀夫研究室）を利用しています
  （CSV版: [openlanguageprofiles/olp-en-cefrj](https://github.com/openlanguageprofiles/olp-en-cefrj)）
- 全問題に一意のIDがあり、結果画面に表示されます。誤りを見つけたらIDでご指摘ください
